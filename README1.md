🌍 Country Explorer App

A responsive web application that allows users to explore countries around the world. Users can search for countries, view detailed information, and filter based on regions.

🚀 Features
🔍 Search countries by name
🌎 Filter countries by region
📊 View country details (population, capital, region, flag, etc.)
📱 Responsive design for all devices
⚡ Fast and interactive UI
🛠️ Tech Stack
React.js – Frontend library
Axios / Fetch API – For API calls
React Router DOM – Navigation
CSS / Tailwind CSS / Bootstrap – Styling

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/your-username/country-explorer.git
2️⃣ Navigate to project folder
cd country-explorer
3️⃣ Install dependencies
npm install
▶️ Running the Application
npm start
App will run on:
👉 http://localhost:3000
🌐 API Used

This project uses a public API to fetch country data:

REST Countries API

Example endpoint:

https://restcountries.com/v3.1/all
